JS Functions |
Concept:

A JavaScript function is a block of code designed to perform a particular task.

To create a JavaScript function, we use the "function" keyword, followed by a name, followed by parentheses () and followed by curly brackets {}:

function name (parameter1, parameter2) {
// code to executed
}
